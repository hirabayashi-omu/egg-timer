# Mind Map Prototype
A modern, dark-themed Mind Map application built with vanilla HTML, CSS, and JavaScript.

## Features
- **Create Nodes**: 
  - `Tab` to add a child node.
  - `Enter` to add a sibling node.
- **Edit**: Double-click any node to edit text.
- **Delete**: Select a node and press `Delete` or `Backspace`.
- **Navigation**:
  - Drag background to pan.
  - Mouse wheel to zoom.
  - Center button to reset view.
- **Save/Load**: Save your map as a JSON file and load it back later.

## Setup
Simply open `index.html` in any modern web browser. No compilation or server required.

## Tech Stack
- HTML5
- CSS3 (Variables, Flexbox, Animations, Glassmorphism)
- Vanilla JavaScript (Recursion for tree layout, SVG for connections)
